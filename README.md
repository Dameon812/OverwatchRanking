# Top 10 Overwatch Characters

## Project Overview
This project is a single-page responsive website created using HTML and CSS. The website ranks my top 10 favorite Overwatch characters while demonstrating basic web design principles such as visual hierarchy, responsive layout, navigation, and interactive elements.

## Features
- Responsive single-page layout
- Navigation links to different sections
- Top 10 Overwatch character rankings
- Dark gaming-themed design
- Hover effects for buttons and ranking items
- External links to the official Overwatch website and the Overwatch Reddit community

## Technologies Used
- HTML5
- CSS3

## Purpose
The purpose of this project is to create a clean and organized interface for Overwatch players to view my personal hero rankings while practicing HTML and CSS skills.

## Responsive Design
The website includes a media query that adjusts the layout for smaller screens by:
- Reducing page padding
- Resizing the main heading
- Scaling images to fit mobile devices

## Future Improvements
If I continue developing this project, I would like to:
- Add images for each hero
- Include short descriptions and abilities for every character
- Add character cards instead of a simple list
- Add simple CSS animations and transitions
